    替换步骤¶

    1. 运行软件并导航到对应页面。
    2. 截图并裁剪至合适尺寸（建议宽度 1200–1600 px，PNG 格式）。
    3. 以相同文件名覆盖 docs/images/ 下对应占位文件。
    4. 运行 .venv/Scripts/python -m mkdocs build --strict 确认构建仍然通过。
    5. 提交更改：git add docs/images/ && git commit -m "docs: add real screenshots"。
