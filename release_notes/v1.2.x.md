# v1.2.x

## v1.2.0

### 质检模块

- **新增质检功能页**：实现完整的挤出力质检（Quality Check）模块，支持按材料规格自动评估挤出力稳定性。
- **实时挤出力读取**：从 hepic_server 的 `extrusion_force_N` 字段实时获取挤出力数据。
- **稳定性评估指示器**：新增三级稳定性指示器（合格 / 警告 / 不合格），实时显示挤出力均值与标准差精确数值，并用半透明色带高亮"优秀"范围，直观呈现数据落点。
- **三栏卡片布局**：质检页采用三栏卡片式布局，分别展示系统信息、挤出力曲线和材料选择与评估结果。
- **两级材料选择**：材料数据库改用 YAML 格式，支持两级选择（材料大类 → 具体规格），方便扩展维护。
- **默认挤出长度**：挤出长度默认值设为 `速度 × 60`（即 1 分钟挤出量），与典型测试时长匹配。

## v1.2.1

### 质检模式增强

- **自动停止质检**：挤出完成后，打印机通过 `M118 STOP_QUALITY_CHECK` 通知主程序，自动点击停止质检，无需手动操作。
- **挤出进度条**：系统信息卡片底部新增进度条，加热完成后开始计时，根据材料挤出速度和长度显示实时进度。
- **状态提示**：进度条上方新增单行状态文字，随 G-code 执行阶段更新：正在加热 → 正在挤出 → 质检完毕，请记录数据。
- **G-code 序列优化**：挤出指令后加入 `M400` 确保动作完成再发信号；序列结束执行 `FIRMWARE_RESTART` 复位固件；质检取点数改为 200 点。
- **移除预期挤出力**：`expected_force` 字段未参与任何评估逻辑，已从 UI、数据库及所有材料配置中删除。

### 序列与录制

- **序列模式自动开始录制**：在动作序列（Job Sequence）模式下，序列启动时自动触发数据录制，无需手动点击录制按钮。

### 打包与分发

- **打包为 Windows 安装程序**：使用 PyInstaller + Inno Setup 将应用打包为标准 Windows 安装程序（Setup.exe），方便分发与部署。
- **新增应用图标**：添加 `.ico` / `.png` 应用图标，已集成至打包脚本和安装程序配置。
